-webkit-transition笔记

时间函数(transition-timing-function)：
- ease：逐渐慢下来
- linear：线性过度
- ease-in：由慢到快
- ease-out：由快到慢
- ease-in-out：由慢到快在到慢
- cubic-bezier：特定的cubic-bezier曲线,所有值需在[0, 1]区域内，否则无效。
